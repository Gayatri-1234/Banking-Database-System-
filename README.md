# Banking-Database-System-
This project is about building a simple banking system using a database to manage customers, accounts, and transactions. It’s designed to track basic account activities like deposits, withdrawals, and balances, and to generate insights about account usage.

Problem Statement:-
The task is to design a database for a banking system that efficiently manages customer information, their accounts, and related transactions. The system should allow easy retrieval of customer account details, track transactions (deposits and withdrawals), and provide insights like total account balance, active accounts, and transaction volumes. The goal is to build a structured database that supports basic banking operations and enables data analysis for customer and account management.

Key Findings:-
•	Total Balance: The system can calculate the total balance of all accounts combined, which gives an overview of the bank's current financial state.
•	Customer Accounts: We can track how many accounts each customer has and find customers who own more than one account.
•	Active Accounts: The system highlights accounts with the most transactions, which can help identify highly active customers.
•	Transaction Volume: Accounts with high transaction volumes (over $1000) can be easily identified, giving insight into more engaged customers

What was achieved?
•	The database can now easily store and manage customer details, account balances, and transactions.
•	Key data views like customer balances and transaction history provide quick access to useful information for analysis.
•	The system can identify active accounts, accounts with no transactions, and customers with multiple accounts.

What’s next?
•	You can build on this system by adding more features like managing loans or integrating automated alerts for low balances or large transactions.
•	For better data analysis, you could add more detailed reports or track account history over longer periods.
This project sets a solid foundation for any further work in a banking system and offers an easy way to manage and analyze customer data effectively.
